## Daily 1: Possible sum (Easy)
Given a list of numbers and a number `k`, return wether any two numbers from the list add up to k.
### Examples
* `PossibleSum([10,15,3,7], 17)` -> `true`, because 10+7 = 17
* `PossibleSum([23,2,17,12], 46)` -> `false`
